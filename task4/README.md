Task 4
The goal of the homework is to reconstruct a CT image from a given sinogram. Please download data files. The file “sinogram.npy” contains sinogram data and the file “CTReferece.npy” contains CT image data. The size of the matrix of sinogram is 512x180 and the size of the matrix of the image is 512x512. Assume that the size of every pixel is 2mm x 2mm. Thus, the size of image is 1.024m x 1.024m. The sinogram was generated by projecting the image using 180 angles with 1 degree interval. As shown in the figure below, every column in the sinogram corresponds to the 1D projection using one angle. Note the rotation direction in the figure. The start angle is 1o and the last angle is 180 o. The rotation of source and detector was conducted counterclockwise with respect to the center of the image.   
1)	Implement the back-projection (BP) algorithm and show the reconstruction result.
2)	Implement the filtered back-projection (FBP) algorithm and show the reconstruction result. 
3)	Write the report to explain all processes, show the results and discuss the results.
